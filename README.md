# Book Store

## Задание
Сделать магазин книг, с возможностью добавлять, удалять, продавать, искать, загружать из/в файл книги используя Python и библиотеку tkinter.
## Ход работы
В ходе выполнения расчётно-графического задания была разработана программа «Book Store» на языке Python с использованием библиотеки tkinter для создания графического интерфейса.
Программа реализована с применением объектно-ориентированного подхода. Основная логика приложения находится в классе BookStore. 

Функциональность программы разделена на несколько групп методов:
•	Основные методы — создание и настройка окна, а также формирование элементов интерфейса;
•	Вспомогательные методы — проверка корректности данных, обновление таблицы и подсчёт общего количества книг;
•	Методы управления — добавление, удаление и продажа книг;
•	Методы поиска — поиск книг по названию и автору;
•	Методы работы с файлами — сохранение данных в файл и загрузка данных из файла;
•	Метод заполнения полей — автоматическое заполнение полей ввода при выборе книги из таблицы.
В результате была создана программа, позволяющая управлять списком книг, выполнять поиск и сохранять данные в файл.
